Starlite

Starlite is a Google home© type helper that runs on a Raspberry Pi© and uses a monitor to display videos, searches, etc. Developed during my time at Gallup in Omaha, Nebraska. Implements usable text-to-speech and speech-to-text methods as well as sofisticated facial recognition for email security.

Artificial Neural Network in Java

An implementation of an Artificial Neural Network in the Java programming language from scratch. Includes self-developed and complete matrix library.
